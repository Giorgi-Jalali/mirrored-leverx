## Overview

### How to launch the app

Browser extension - "Cross Domain - CORS" (without it there is a CORS error)

- yarn i

It needs two terminals

- yarn start
- yarn run server (for json-server)

Production

- yarn build
- yarn start-pro
- yarn run server (for json-server)
